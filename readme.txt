Steganography wav

Requirements:
    Python 3.x

General usage:
    For using program just run main.py in folder "python files" or run script in console:
    python main.py [options] [arguments] for windows
    python3 main.py [options] [arguments] for linux-based OS
    run script with parameter "-h" for showing help


Environment:
"python files" is directory with .py files, includes main.py
"python files/stegano_logs.log" is log file
"text files" is directory with files for testing steganography
"unpack_dir" is default directory in which files will be unpacked
"wav files" is directory with wav files for testing steganography
